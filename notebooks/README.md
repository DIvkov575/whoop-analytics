# Whoop Sleep Quality — Causal Analysis Notebook

## Quick Start

1. Register a Whoop developer app at https://developer.whoop.com
   - Set redirect URI to `http://localhost:9876/callback`
2. Copy `.env.example` to `.env`, fill in `WHOOP_CLIENT_ID` and `WHOOP_CLIENT_SECRET`
3. Run: `jupyter notebook notebooks/sleep_causal_analysis.ipynb`
4. Execute cells top-to-bottom — the first section will open your browser for OAuth login
